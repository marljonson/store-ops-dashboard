# Store Ops Impact Dashboard

### Technologies used
Python, Pandas, Excel

### Files
* `notebooks/analysis.ipynb`
* `data/store_data.csv`

### Objective
Measure impact of new store initiatives on sales, labor efficiency, and operator workload.

### Key Findings
- Stores adopting initiatives saw +15% average sales growth
- Labor efficiency improved by 10%
- Adoption was high across 3 pilot stores

### Recommendations
- Roll out initiative to remaining stores
- Provide adoption support & training for operators
